2. Cálculo del IRPF paso a paso

Paso 1: Rendimiento neto previo

rn = bruto – cotizaciónSS

Art. 18 LIRPF

Paso 2: Aplicar reducción art. 20

rn_reducido = rn – reducción_art20

Art. 20 LIRPF

Paso 3: Rendimiento neto

rneto = rn_reducido – gastos_fijos(2.000 €)

Art. 19 LIRPF — Gastos deducibles: 2.000 € anuales.

Paso 4: Base liquidable general

baseLiquidable = rneto
⚠ IMPORTANTE: El mínimo personal NO se resta de la base liquidable. Art. 63 LIRPF — «El importe del mínimo personal y familiar minorará la cuota íntegra.»

Paso 5: Cuota íntegra

cuotaÍntegra = Σ (baseTramo × tipoCombinado) donde para cada tramo [limiteInferior, limiteSuperior, tipo]: baseTramo = min(baseLiquidable, limiteSuperior) – limiteInferior (solo si baseLiquidable > limiteInferior)

Art. 63.1 LIRPF + Leyes autonómicas

Paso 6: Reducción por mínimo personal y familiar

cuotaMínimo = Σ (baseMínimo × tipoCombinado) donde: baseMínimo = min(mínimoTotal, anchoTramo, baseDisponible) (recorriendo los tramos de la escala progresiva)

El mínimo se aplica como reducción de cuota calculada sobre los tipos marginales reales de cada tramo, no como un tipo fijo. Un contribuyente con renta alta se beneficia menos del mínimo que uno con renta baja.

Paso 7: Cuota líquida

cuotaLíquida = cuotaÍntegra – cuotaMínimo

Paso 8: Deducción por SMI (Art. 80 LIRPF)

Si bruto ≤ 16.576: dedSMI = 340 Si bruto ≤ 18.276: dedSMI = 340 – 0,20 × (bruto – 16.576) Si bruto > 18.276: dedSMI = 0

Paso 9: Cuota resultante

cuotaResultante = cuotaLíquida – dedSMI

Paso 10: Límite de retención (Art. 101.1 LIRPF)

límiteRetención = max(0, (bruto – 15.876) × 0,43) irpfFinal = min(cuotaResultante, límiteRetención)

3. Desglose estatal / autonómico

CCAA comunes

// Cuota estatal baseEstatal = baseLiquidable cuotaEstatal = Σ (baseTramo × tipoEstatal) cuotaLiqEstatal = cuotaEstatal – reducciónMínimoEstatal cuotaResEstatal = cuotaLiqEstatal – (dedSMI × proporciónEstatal) irpfEstatal = min(cuotaResEstatal, límiteRetención) // Cuota autonómica irpfAutonómica = irpfFinal – irpfEstatal

CCAA forales (Navarra y País Vasco)

irpfAutonómica = irpfFinal irpfEstatal = 0

No hay desglose — escalas completas propias.

4. Ejemplos de verificación

Verificación cruzada contra simuladores oficiales. Todos los ejemplos son reproducibles.

Ejemplo 1: Contribuyente individual, 35.000 € bruto — Madrid

bruto = 35.000 cotizaciónSS = min(35.000, 58.914) × 0,0635 = 2.222,50 € rn = 35.000 – 2.222,50 = 32.777,50 € reducción_art20 = 0 (rn > 19.747,50) rneto = 32.777,50 – 2.000 – 0 = 30.777,50 € baseLiquidable = 30.777,50 € // CUOTA ESTATAL (escala Art. 76 LIRPF) 12.450 × 0,095 = 1.182,75 (20.200 – 12.450) × 0,12 = 930,00 (30.777,50 – 20.200) × 0,15 = 1.586,63 cuotaEstatal = 3.699,38 € // CUOTA AUTONÓMICA Madrid (PDF Hacienda pág. 47) 13.362,22 × 0,085 = 1.135,79 (19.004,63 – 13.362,22) × 0,107 = 603,72 (30.777,50 – 19.004,63) × 0,128 = 1.506,93 cuotaAutonómica = 3.246,44 € cuotaÍntegra = 3.699,38 + 3.246,44 = 6.945,82 € // Reducción mínimo (5.550 €) cuotaMinEstatal: 5.550 × 0,095 = 527,25 € cuotaMinAut: 5.550 × 0,085 = 471,75 € cuotaMínimoTotal = 999,00 € cuotaLíquida = 6.945,82 – 999,00 = 5.946,82 € dedSMI = 0 (bruto > 18.276) límiteRetención = (35.000 – 15.876) × 0,43 = 8.223,32 € irpfFinal = min(5.946,82, 8.223,32) = 5.947 € neto = 35.000 – 2.222,50 – 5.947 = 26.830,50 € tipoEfectivo = 5.947 / 35.000 = 16,99%

Resultado: 26.831 € neto · 5.947 € IRPF · 17,0% tipo efectivo
